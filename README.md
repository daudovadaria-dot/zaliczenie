# Zoo Online

## Opis projektu
Zoo Online to prosty projekt strony internetowej przedstawiający różne zwierzęta wraz z ich opisami i zdjęciami. Celem projektu jest nauka podstaw pracy z HTML oraz Git i GitHub.

## Funkcjonalności
- prezentacja zwierząt
- zdjęcia zwierząt w katalogu img
- krótkie opisy gatunków
- struktura projektu oparta o HTML

## Struktura projektu
- index.html – główna strona projektu
- img/ – folder ze zdjęciami zwierząt

## Technologie
- HTML5
- Git
- GitHub
